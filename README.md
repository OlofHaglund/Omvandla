# Omvandling

A Vue + Vite app for converting ingredient measurements using density values from `ingredienser.json`.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Data source

- `public/ingredienser.json`
- Structure: ingredient name as key, with `densitet` in `kg/dm3`.
